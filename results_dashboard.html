Research Readout
The project asks a direct quant-research question: if a model writes the forecasting code itself, does that code earn its place once the same out-of-sample scoring harness is applied to everybody?

The answer from the saved repository outputs is mixed. The project now contains a full cross-platform attempt universe, but only a subset has been fully rescored. NVIDIA NIM currently dominates the scored book because those outputs were run through the evaluation harness end to end, while Ollama and Hugging Face currently appear as generated or failed entries in the saved repository history.


That means this page is not pretending every row is equally mature. It is deliberately transparent: scored models, generated notebooks awaiting rescoring, gated failures, and empty outputs are all shown together so the project presentation matches the actual repository record.
