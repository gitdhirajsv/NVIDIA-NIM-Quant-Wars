Research Readout
The project asks a direct quant-research question: if a model writes the forecasting code itself, does that code earn its place once the same out-of-sample scoring harness is applied to everybody?

The answer from the committed repository reports is clearer now. The repo contains one rescored cross-platform report book: 63 NVIDIA reports, 11 Ollama reports, and 6 Hugging Face reports, all evaluated under the same harness, plus a small set of empty NVIDIA notebooks that are shown explicitly instead of being hidden.


That makes this page a report surface, not a marketing surface. It is deliberately transparent about what was scored, what was empty, which venue produced the strongest models, and how often compact code still held up out of sample once the notebooks were rescored inside this repo.
