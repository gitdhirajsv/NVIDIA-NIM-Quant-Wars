## 🔬 Methodology

**Data:** 100,000 rows sampled from real Jane Street partitioned parquet dataset  
**Target:** `responder_6` — anonymized 8-day forward return  
**Split:** 80% train / 20% test (random_state=42)  
**Metric:** Mean Squared Error (MSE) — lower = better  

**Evaluation approach:**
1. Extract Python code from each `.ipynb` notebook
2. Detect feature engineering pattern (quantile flag, global vs per-date)
3. Extract XGBoost hyperparams via regex from model's own code
4. Train and evaluate with standardized split
5. Compute MSE, RMSE, and R²
